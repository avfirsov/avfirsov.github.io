Урок 29. Реакции диспропорционирования и сопропорционирования

Среди ОВР, отдельно выделяют реакции, где окислителем и восстановителем является один и тот же элемент. Вот пример такой реакции:

H2S + SO2 ⇒ S + H2O

Диспропорционирование и сопропорционирование

ОВР одного элемента протекают по одной из двух схем:

  1. Либо элемент переходит из двух степеней в одну (как в примере выше) - такие реакции называются сопропорционированием.

  2. Либо элемент переходит из одной степени окисления в две разных новых степени окисления - такие реакции называются диспропорционированием.

Пример реакции диспропорционирования:

P + NaOH + H2O ⇒ PH3 + NaH2PO2

Как уравнивать

Такие реакции уравниваются легко: по балансу находим коэффициенты для веществ с частицами, которые не повторяются в балансе, остальные коэффициенты находим подбором. В примере с серой, по балансу мы находим коэффициенты только для H2S и SO2, а остальные коэффициенты подбираем:

2H2S + 1SO2 ⇒ S + H2O

Надо подобрать S и H2O

Если попытаться применить множитель баланса в качестве коэффициента перед S(0), возникает неопределенность: S(0) присутствует и в первой строке баланса с множителем 2, и во второй строке баланса, с множителем 1. Какой из множителей выбрать в качестве коэффициента - непонятно. Поэтому поступим проще и подберем коэффициент перед S, а заодно и перед водой:

2H2S + 1SO2 ⇒ 3S + 2H2O

Аналогично уравнивается вторая реакция. По балансу находим коэффициенты перед PH3 и Na2HPO2:

P + NaOH + H2O ⇒ 1PH3 + 3NaH2PO2

Остальные коэффициенты легко находим подбором:

4P + 3NaOH + 3H2O ⇒ 1PH3 + 3NaH2PO2

Задачи

Расставьте коэффициенты в следующих уравнениях:

FeCl3 + Fe ⇒ FeCl2
NO2 + H2O ⇒ HNO2 + HNO3
NaNO2 + (NH4)2SO4 ⇒ Na2SO4 + N2 + H2O
S + NaOH ⇒ Na2S + Na2SO3 + H2O
H2S + H2SO4 ⇒ SO2 + H2O
S + H2SO4 ⇒ SO2 + H2O
NaH + H2O ⇒ NaOH + H2
Cl2 + KOH ⇒ KCl + KClO + H2O
Br2 + NaOH ⇒ NaBr + NaBrO3 + H2O
